# FinPibble

App mobile de finanças pessoais criado com React Native e Expo.

## Stack

- Expo
- React Native
- JavaScript
- React Navigation
- StyleSheet nativo
- lucide-react-native

## Rodando o projeto

Requer Node.js 20.19.4 ou superior.

```bash
npm install
npm start
```

Use o QR Code do Expo Go ou os comandos `npm run android` e `npm run ios`.
